# turbo-hearts-scores

## Developing
1. `yarn`
2. `yarn build`

### Run the server
1. `cd projects/server`
2. `yarn dev`
If you want datas, grab a DB backup from https://scores.anti.run/backups/ and replace projects/server/dev.sqlite3

### Run the front end
1. `cd projects/app`
2. `yarn dev
3. `open localhost:1234`
